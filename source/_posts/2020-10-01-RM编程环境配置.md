## 安装VS 2019

前面所有步骤略过。

安装位置最好默认安装，然后因为我们只需要用VS 2019在windows环境下的C++，所以只勾选使用C++的桌面开发。然后点击安装，喝杯咖啡等待下载安装。
![勾选](https://raw.githubusercontent.com/cyicz123/Picture/main/img/20201002101918.png)

一杯咖啡后安装完毕，点击重启计算机。

## 编写自己的第一个程序Hello world

启动下载好的VS 2019并且按照提示登陆自己的微软账号。

 选择创建新项目

![创建新项目](https://raw.githubusercontent.com/cyicz123/Picture/main/img/20201002104421.png)

选择创建空项目或者控制台应用都可以。这里我选择的是空项目。

![选择空项目](https://raw.githubusercontent.com/cyicz123/Picture/main/img/20201002104453.png)

在右侧源文件选项中右键点击新建文件，或者Ctrl+Shift+A直接快捷键新建文件。

![](https://raw.githubusercontent.com/cyicz123/Picture/main/img/20201002105230.png)

选择C++文件，名称最好改为英文名称，这里我改为main.cpp

![](https://raw.githubusercontent.com/cyicz123/Picture/main/img/20201002105256.png)

输入代码

```c++
#include<iostream>
using namespace std;
int main()
{
    cout<<"Hello world!"<<endl;
    //可以在双引号里写自己想输出的文字。
    //所有符号一定要是英文输入法里的符号。；和;是不同的。
    return 0;
}
```

F5执行程序
