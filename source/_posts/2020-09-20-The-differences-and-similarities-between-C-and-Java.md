## the similarities 

Java and C++ are object-oriented language which both go in with object-oriented mentality(encapsulation,inherit,polymorphism).Because object-oriented has many very good features,such as inherit,polymorphism,etc,Java and C++ have excellent reusability.

Java and C++ not only have the most similarity in object-oriented design mentality,but also are similar in the syntax.

They have same keywords,such as `if for while const new`.Although they have too many similar in syntax and object-oriented mentality,there are lots of different between Java and C++ in kernel.
